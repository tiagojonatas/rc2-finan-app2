# RC2 Finance Brand Assets

## Files
- `rc2-finance-horizontal.svg`: horizontal logo (navbar/login header)
- `rc2-finance-icon.svg`: icon/monogram (avatar, app icon area)
- `rc2-finance-favicon.svg`: favicon SVG

## Palette
- Green: `#22C55E`
- Teal: `#14B8A6`
- Blue: `#0EA5E9`
- Neutral text (horizontal): `#0F172A`

## Suggested usage
- Navbar: horizontal logo with `h-8` to `h-10`
- Login: horizontal logo or icon with `h-10` to `h-14`
- Favicon: `rc2-finance-favicon.svg`
